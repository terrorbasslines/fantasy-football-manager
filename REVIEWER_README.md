# Fantasy Football Text Manager Review Build

This folder contains a review-friendly install package for the MVP.

## Windows Quick Start

1. Install Node.js LTS from https://nodejs.org if it is not already installed.
2. Double-click `install-windows.bat`.
3. Double-click `run-windows.bat`.
4. Open the local URL printed by the app, usually `http://127.0.0.1:5173`.

## Browser Install

After the app is running, supported browsers can install it as an app:

- Chrome or Edge: open the address bar install button, or use browser menu > Apps > Install this site as an app.
- The installed app still stores saves locally in that browser profile.

## What To Review

- Start a career as Nickelodeon FC or Hondsberg FC.
- View squad and player profiles.
- Import or remove miniface images.
- Simulate fixtures and read match commentary/report output.
- Check the league table after matchday simulation.
- Save and load career progress in Settings.
- Export/import the editable JSON database in Database Editor.

## Notes

This is not a 3D football game. It is a text-based football management MVP with fictional clubs and players only.
